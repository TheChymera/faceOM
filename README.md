#faceOM

Python and MATLAB scripts for data processing and analysis written for the [“Neuronal Correlates of Occulometric Parameters in Face Recognition”](https://github.com/TheChymera/masterarbeit) research project.

DEPENDS on:

* numpy
* scipy
* matplotlib
* pandas
* MATLAB
